Usage Guide

URL Context

Only URLs that return textual results are supported for now. HTML is disabled by design to incentivize users to improve their context.

Recommended Contexts

xymake.com - X threads uithub.com - GitHub context openapisearch.com - APIs

Result Format

Any lmpify URL is available as HTML, JSON, or Markdown. Browsers return HTML by default while developer tools or APIs like curl or fetch default to markdown.

You can control the output by appending .json/.md/.html or by specifying an accept header.

It's also possible to get a subset of the markdown through ?key=result|context|prompt

Examples

JavaScript fetch (returns markdown):
fetch("https://lmpify.com/httpsuithubcomj-m8tfk00").then(res=>res.text())
Terminal curl (returns markdown):
curl https://lmpify.com/httpsuithubcomj-m8tfk00
Browser (returns HTML UI):
https://lmpify.com/httpsuithubcomj-m8tfk00
Get markdown directly:
https://lmpify.com/httpsuithubcomj-m8tfk00.md
Get JSON format:
https://lmpify.com/httpsuithubcomj-m8tfk00.json
Get only the result:
https://lmpify.com/httpsuithubcomj-m8tfk00.md?key=result

Chat Completions

Coming soon

Every prompt is made available as chat completions endpoint at POST https://lmpify.com/[id]/chat/completions.

This means you can use it by setting the base path in the OpenAI SDK (or other ChatCompletion SDKs) to https://lmpify.com/[id], e.g. https://lmpify.com/httpsuithubcomj-m8tfk00.

This will use the context as system prompt and the prompt as a first 'model' message in the back.

To use a model other than the default model, you can specify the model parameter. For available models, check models.json

Getting Your API Key

You can get your API key by going to the developer console in your browser, and find the access_token value in your cookies storage.

This serves as authentication to any API. There's currently no scopes or ability to rotate your key, so be careful, this key is meant to be private. If it gets compromised, all your balance may be used by third parties.

npx mdapply CLI

You can use mdapply to apply a response output to your local filesystem.

Just run:

curl https://lmpify.com/[id]?key=result -o apply.md && npx mdapply ./apply.md
