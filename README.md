# `contextarea.js` - textarea for context-building

> [!IMPORTANT]
> 🚧 This is a WIP and the below is not done yet ⚠️

Requirements:

1. [pastebin](./pastebin) for OS-provided text or binary files (via copy or drag/drop) (https://pastebin.contextarea.com/worker.js)
2. [context](./context) for fetching/caching any URL (or string with urls) into the belonging metadata to build up a beautiful UI (https://context.contextarea.com/worker.js)
3. [root](.) for js library to enhance any textarea with the above, easily embeddable (https://contextarea.com/lib.js)
4. [demo](./demo) that brings it all together in a standalone html (https://demo.contextarea.com)
