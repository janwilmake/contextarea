Prompt It Buttons

Easy to implement buttons for your website to send content to lmpify

Getting Started

Add beautiful "Prompt It" buttons to your website to help users send content directly to lmpify. These buttons allow you to pre-fill prompts and URLs, making it simple for users to analyze your content.

1. Add the Script

First, include the button-widget.js script in your HTML:

<script src="https://lmpify.com/js/button-widget.js"></script>

2. Add a Button

Add a button element with the data-lmpify-button attribute:

<button data-lmpify-button data-size="md" data-style="basic-3d"></button>

Available Options

Customize your buttons with these data attributes:

Attribute Options Description
data-size "small", "medium", "large", "xl" Button size (default: "medium")
data-style "style1", "style2", "style3", "style5" Visual style (default: "style1")
data-text "true", "false" Show "prompt it" text (default: "true")
data-url URL string URL to be analyzed (optional)
data-prompt Text string Pre-filled prompt text (optional)

Style Reference

"style1": Basic 3D button
"style2": Gradient button
"style3": Button with border
"style5": 3D button with shadow

Advanced Examples

Pre-filled Prompt

Create a button that opens lmpify with a specific prompt already filled in:

<button data-lmpify-button data-size="large" data-style="style5" data-prompt="Summarize this content and identify the key points" ></button>

Analyzing a Specific URL

Create a button that sends a specific URL to be analyzed:

<button data-lmpify-button data-size="large" data-style="style2" data-url="https://example.com/content-to-analyze" ></button>

URL + Specific Prompt

Combine both URL and prompt for a complete experience:

<button data-lmpify-button data-size="xl" data-style="style1" data-url="https://example.com/article" data-prompt="Summarize this article in 3 bullet points" ></button>
